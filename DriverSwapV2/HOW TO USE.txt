🚀 QUICK START:
   Double-click "DriverSwap.bat" to start the tool!

📁 HOW TO USE:
   • Select Root Folder
   • Select Source Driver
   • Choose Drivers to swap
   • Swap and enjoy


📋 WHAT EACH FILE DOES:
   ✅ DriverSwap.bat        ← CLICK THIS to start the tool
   ⚙️ MainCode.vbs          ← Technical file (don't touch)
   🎨 Icon.ico              ← App icon
   📄 Directory.json ← Settings (auto-created)

🎯 FEATURES:
   • Select multiple drivers to replace at once

💡 TIPS:
   • The tool remembers your AC folder location
   • You can edit the AC path directly in the text field
   • Select "All" or individual drivers to replace
   • Choose any .kn5 file or folder as your source
   • Find Driver Models Here --->   https://proabrakadabragaming.blogspot.com/2023/08/130-assetto-corsa-driver-models-in.html

🆘 NEED HELP?
   • Make sure both DriverSwap.bat and MainCode.ps1 are in the same folder
   • Run as administrator if you get permission errors
   • Check that your Assetto Corsa path is correct
